Donor Experience
A donor portal as good as your
best relationship
Your donors log in to a branded, personal giving dashboard — where their funds, history, and giving tools live. Fund status, statements, grant history, and the Charitable Catalog are all one click away. The portal deepens donor engagement between major gift conversations.
✓
Fund balance and available-to-spend — live from back-office
✓
Historic fund statements — HTML with drill-down or uploaded PDF
✓
Grant recommendation worksheet — bulk recommendations, pre-populated from history
✓
Board vote capability for majority/unanimous fund types
✓
Gift entry — credit card, check, wire, or securities transfer
✓
Philanthropic interest preferences that filter the Charitable Catalog
✓
Investment performance page linked by fund type
✓
Targeted content articles based on donor interests — automated charitable guidance
Fund Dashboard
Donors see real-time balances, available-to-spend, and a full history of gifts and grants — all synced directly from your fund accounting system.
Live Data
Grant Recommendation Worksheet
Donors make multiple grant recommendations at once from a pre-populated worksheet of previous grantees — or discover new nonprofits from the Charitable Catalog.
Self-Service
Personalized Content
Articles and funding opportunity notices are automatically surfaced based on each donor's self-reported philanthropic interests — keeping donors engaged between major gifts.
Targeted
Multi-Fund View
Donors associated with multiple funds see all funds in a single dashboard — with per-fund drill-down to balances, statements, and grantmaking tools.
Multi-Fund
Competitive Grantmaking
The most configurable grant
system in the industry
GiftingNet's grant lifecycle system handles every phase from application to final payment — for discretionary competitive grants, RFP processes, and scholarship programs. Every form component, qualification rule, and review workflow is configurable to your foundation's specific process.
✓
One or two-phase applications — LOI followed by full proposal
✓
Pre-qualifying questions that gate application access
✓
Limit applications by org type, profile completion, or invitation only (RFP)
✓
Custom form builder — questions, budgets, matrices, document uploads, board lists
✓
Assign review panels with custom evaluation forms — staff or committee
✓
Grantee progress reports — custom forms, published at set intervals
✓
Grant agreements and payment plans — grantees review, print, and sign online
✓
Syncs to back-office accounting software for posting and payment processing
Application Form Builder
Build grant applications from reusable components: instruction pages, pre-qualifying questions, budget forms, matrix data collectors, document uploads, board lists, and custom question panels.
No-Code Builder
Reviewer Dashboard
Panel members log in to see assigned applications, complete custom evaluation forms, and submit reviews. Dockets are created directly from GiftingNet data.
Panel Management
Due Diligence System
Required documents (990s, IRS letters) are managed through a structured review, approval, and update workflow — with document expiry reminders and barcode-based FAX ingestion.
Compliance
Progress Reporting
Custom progress report forms are assigned to grants and automatically published to grantees at pre-selected intervals. Past-due reminders sent to both grantee and staff.
Automated
Charitable Catalog
Your community's giving
marketplace
The Charitable Catalog is an illustrated, searchable directory of vetted nonprofits and charitable programs — maintained by the organizations themselves. It powers donor grant recommendations, serves as the RFP invitation pool, and functions as an open community resource when you choose to share it publicly.
✓
Organization profiles — mission, history, interest areas, geographic reach, budgets, board
✓
Program-level profiles for specific charitable projects — earmarkable by donors
✓
Keyword search, interest area, geography, program type, and capital campaign calendar
✓
Catalog filtered to each donor's self-reported philanthropic interests
✓
Gift Basket — shopping cart for grant recommendations and outright contributions
✓
Foundation controls which catalog components are visible per user type
✓
Org Group Icons — visual identifiers for any organization category throughout the catalog
✓
NTEE area codes supported — maps to standard nonprofit taxonomy
Donor-Filtered Discovery
Each donor's catalog view is filtered by their self-reported philanthropic interests — surfacing the most relevant nonprofits and programs for their giving priorities.
Personalized
Nonprofit Self-Service Profiles
Grant seekers maintain their own organization and program profiles through the portal. Foundation staff reviews and approves profile changes before they appear in the catalog.
Self-Maintained
Capital Campaign Tracking
Capital campaigns have dedicated catalog search tools — donors can discover and contribute to capital projects as a distinct giving category separate from operating support.
Capital Campaigns
Community Resource
The catalog can be opened to the general public as an informational resource — or kept private for donors and advisors only. Your call, your configuration.
Configurable Access
API & Integrations
Open, real-time sync with
your back office
GiftingNet's web service API is open to any third-party system and provides real-time, bidirectional synchronization with your fund accounting software. Every gift, grant, fund statement, and contact update flows seamlessly between your back office and the donor-facing platform.
✓
Bidirectional sync: contacts, funds, gifts, grants, statements, investment performance
✓
Compatible with FIMS, FoundationPower, and any third-party XML client
✓
Blackbaud SKY API integration — Raiser's Edge NXT and Financial Edge NXT
✓
Orion Advisor Solutions and Envestnet platform integrations
✓
Salesforce CRM, Black Diamond Trust, Greenhill, NetSuite
✓
API Transfer Report monitors all data jobs — find by job number or record ID
✓
Alternative text file import for all data categories — no API required
✓
Hosted on Google Cloud / AWS — Level 1 PCI certified, nightly backups
Back-Office Sync
Fund values, gift confirmations, grant postings, and investment performance flow from your accounting system into GiftingNet — keeping every donor's portal view accurate in real time.
Real-Time
Advisor Platform Integration
